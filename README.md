練習２
x<-1　y<-2 z<-x+y 入力


| コード      | 意味                  | 結果         |
| -------- | ------------------- | ---------- |
| `a <- 1` | 変数 `a` に 1 を代入する    | ―（代入のみ）    |
| `a + 2`  | `a` に 2 を足して結果を表示する | `3`（1 + 2） |

| 操作内容 	| Pythonの書き方	| Rの書き方 |	結果 |
| --------- | -------------- |  -------  | -----  |
|足し算	|1 + 2 |	1 + 2 |	3 |
|引き算 |	5 - 3|	5 - 3|	2 |
|掛け算 |	4 * 2	|4 * 2 |	8 |
|割り算 |	8 / 2	|8 / 2	|4.0|
|整数除算|	8 // 3|	8 %/% 3	|2|
|剰余（あまり）|	8 % 3|8 %% 3|	2|
|累乗（べき乗）|	2 ** 3 or pow(2, 3)	|2^3 または pow(2, 3)|	8|
